# 🐱 小猫咪工资计算器 — 部署指南

## 📦 pak 文件夹内容
```
pak/
├── 小猫咪工资计算器.html   # 主文件（含所有图标，无外部依赖）
├── manifest.json           # PWA 配置（图标、名称、全屏）
├── sw.js                   # Service Worker（缓存 + 自动更新）
└── icons/                  # App 图标（需自行放入，见下方说明）
    ├── icon-192.png
    └── icon-512.png
```

## 🚀 一次性部署步骤（约 10 分钟）

### 第一步：准备图标
- 准备一张 **1024×1024** 的 PNG 图标
- 压缩为 192×192 → 存为 `icons/icon-192.png`
- 压缩为 512×512 → 存为 `icons/icon-512.png`
- 可用工具：https://www.pwabuilder.com/imageGenerator

### 第二步：上传到 GitHub
1. 打开 https://github.com → 新建仓库（如 `salary-cat`）
2. 把整个 `pak` 文件夹里的内容上传（拖拽即可）
3. 确保仓库设为 **Public**

### 第三步：部署到 Vercel（免费）
1. 打开 https://vercel.com，用 GitHub 登录
2. 点击 "Import Project" → 选你刚建的仓库
3. 直接点 Deploy（无需任何配置）
4. 部署完成后会得到一个链接，如：
   `https://salary-cat.vercel.app/小猫咪工资计算器.html`

### 第四步：用户安装（iOS）
1. 用 **Safari** 打开上面的链接
2. 点击底部「分享」按钮 □↑
3. 选择「添加到主屏幕」
4. 点「添加」→ 桌面出现 App 图标 ✅

## 🔄 日常更新流程（一键发布）

```
修改 salary-widget-standalone.html
    ↓
复制到 pak/小猫咪工资计算器.html
    ↓
修改 sw.js 里的 CACHE_VERSION（如 v1.0.1）
    ↓
git add . && git commit -m "update" && git push
    ↓
Vercel 自动部署（约 30 秒）
    ↓
所有用户下次打开 App 自动拉取新版本 ✅
```

---

## ℹ️ 注意事项
- 用户数据存在各自手机的 localStorage，**不会因更新丢失**
- iOS 需用 Safari 才能「添加到主屏幕」，Chrome 不支持
- 首次部署后发给用户链接即可，后续更新用户无需任何操作
